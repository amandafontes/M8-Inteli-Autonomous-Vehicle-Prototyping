<h2>Atividade Prática | P1 do Módulo 8</h2>

A atividade caracteriza-se por um script Python que cumpre o papel de um rule-based chatbot. Por meio de expressões regulares pré-definidas, o código é capaz de interpretar o comando fornecido pelo usuário e relacioná-lo à sua respectiva intenção. Desse modo, o script retorna a informação que o usuário deseja obter.

O código corresponde ao arquivo <code>chatbot.py</code>, localizado no presente diretório.
